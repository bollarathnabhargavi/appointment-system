# Smart Clinic Appointment System

## Overview

Smart Clinic Appointment System is a full-stack web application that enables patients to book appointments with doctors, manage schedules, and view appointment details efficiently.

## Features

* User Registration and Login
* JWT Authentication and Authorization
* Doctor Management
* Appointment Booking and Scheduling
* Patient Management
* Analytics Dashboard
* Secure REST APIs
* Responsive User Interface

## Tech Stack

### Backend

* Java
* Spring Boot
* Spring Security
* JWT Authentication
* Maven

### Frontend

* React.js
* Vite
* Tailwind CSS
* Axios

### Database

* MySQL

## Project Structure

appointment-system/

* Backend (Spring Boot)
* Frontend (React + Vite)

## How to Run

### Backend

1. Open the backend project.
2. Configure database settings in application.properties.
3. Run:

mvn spring-boot:run

### Frontend

1. Navigate to frontend folder.
2. Install dependencies:

npm install

3. Start the application:

npm run dev
